# Renomeador de Arquivos

Aplicação em Python com interface gráfica para renomear arquivos em lote de forma rápida e segura.

##  Funcionalidades

- Selecionar pasta
- Definir nome base
- Renomear múltiplos arquivos automaticamente
- Interface gráfica amigável

##  Tecnologias

- Python
- Tkinter

## ▶️Como executar

1. Clone o repositório
2. Execute:

```bash
python renomeador_gui.py
